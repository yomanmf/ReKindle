Telegram for ReKindle

A quiet, text-first Telegram client powered by your Beeper Desktop.

  1. Install Beeper Desktop and connect your Telegram account.
  2. In Beeper, open Settings, then Integrations, and enable Desktop API.
  3. Create an approved connection token and enable Remote Access for your Kindle.
  4. Use an HTTPS tunnel address when ReKindle is opened over HTTPS.
This page uses HTTPS. Most browsers will block a plain HTTP Beeper address; use an HTTPS tunnel.
The address and token stay in this browser. ReKindle does not upload them to Firebase or Yandex Cloud.
